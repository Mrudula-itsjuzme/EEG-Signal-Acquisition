# EEG Signal Acquisition and Analysis

A research-planning repository for EEG signal acquisition, preprocessing, literature review, and experimental design.

The project focuses on collecting EEG data under controlled conditions and studying how brain activity changes across cognitive, physical, and environmental states.

---

## Project goals

- design an EEG signal acquisition workflow
- study EEG behavior across different experimental conditions
- identify frequency-band patterns such as alpha, beta, theta, and delta activity
- understand noise, artifacts, and preprocessing challenges
- prepare a foundation for BCI, neurofeedback, and cognitive-state monitoring research

---

## Why this matters

EEG signals are sensitive, noisy, and highly dependent on acquisition quality. A strong acquisition protocol is the backbone of any reliable EEG-based analysis system.

This repository documents the research groundwork before moving into full data collection and model development.

## Literature review

The README contains a curated literature review covering EEG acquisition, preprocessing, wearable hardware, artifact removal, source localization, machine learning, emotion recognition, seizure detection, depression assessment, Alzheimer’s detection, and BCI-focused wearable design.

The papers span review articles, applied systems, and recent EEG+AI work, making the repository useful as a research reference as well as a project log.
